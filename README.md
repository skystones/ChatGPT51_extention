# ChatGPT Model Selector

ChatGPT の新規チャット作成時に、指定したモデル (例: GPT-5.1 Thinking) を自動的に選択する Chrome 拡張です。

## インストール方法 (ローカル読み込み)

1. このリポジトリをローカルに配置します。
2. Chrome で `chrome://extensions/` を開きます。
3. 右上の **デベロッパーモード** をオンにします。
4. **パッケージ化されていない拡張機能を読み込む** をクリックします。
5. このリポジトリのフォルダを選択します。

インストール後、`https://chatgpt.com/` を開くと自動でモデル選択が実行されます。

## 使い方

- 基本設定はそのまま動作します。
- モデル名やセレクタを変更したい場合は、拡張機能の詳細画面から **オプション** を開いて編集してください。

## 設定項目

- **選択するモデル名**: メニューに表示されるモデル名と一致させます。
- **再試行間隔 (ms)**: 要素が表示されるまでの再試行間隔。
- **各セレクタ**: UI の構成が変わった場合に調整してください。

## 注意事項

- chatgpt.com の UI 変更で動かなくなる可能性があります。
- その場合はオプション画面でセレクタを更新してください。
